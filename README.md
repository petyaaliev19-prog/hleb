# Хлебушек

`Хлебушек` — веб-сервис предзаказа свежего хлеба и выпечки с самовывозом и оплатой на месте.

## Что уже есть

1. Каталог пекарен.
2. Страница отдельной пекарни с ассортиментом и слотами самовывоза.
3. Оформление предзаказа без онлайн-оплаты.
4. Лента заказов.
5. Панель пекарни со сводкой по позициям на день.
6. Смена статусов заказа в панели пекарни.
7. Локальная SQLite база через встроенный `node:sqlite`.

## Текущий стек

1. `Next.js` для приложения.
2. `React` для интерфейса.
3. `node:sqlite` для локального persistence-слоя.
4. `node --test` для юнит-тестов.

## Основные файлы

1. `app/` — страницы и API.
2. `components/` — UI-компоненты.
3. `lib/db-repository.js` — работа с SQLite.
4. `lib/order-status.js` — правила статусов заказа.
5. `tests/` — юнит-тесты текущей бизнес-логики.
6. `work/hlebushek.db` — локальный файл базы данных.

## Команды

1. `npm.cmd run dev` — локальный dev-сервер.
2. `npm.cmd test` — запуск тестов.
